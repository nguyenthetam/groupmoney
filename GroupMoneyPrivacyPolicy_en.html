Privacy Policy

Effective date: Aug, 10, 2025

Group Money App is committed to protecting user privacy. This policy explains how we collect, use and protect your personal information when using the application.

1. Information we collect

The Group Money app stores all data in the app's memory on the user's device. The data export function creates a text file (.CSV) that is also placed on the user's device. The user can then choose to share it with others proactively. This means that we (the app's author) do not collect any information or data from the user.

2. How we use information

We (the app's author) do not use any user information or data at all.

3. Sharing information

We (the app's author) absolutely do not share user information or data. As mentioned above, the function of exporting text files and sharing with others can only be done if the user really wants it.
